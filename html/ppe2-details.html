PPE 2
SITE M2L 2.0

Évolution du portail de la Maison des Ligues vers le Web Dynamique et mise en place d'une architecture MVC.

MISSION 1 : PASSAGE AU DYNAMIQUE

PHP / SQL / DOCUMENTATION

La première étape consistait à transformer le site statique en un site dynamique. J'ai rédigé les scripts de création de la base de données et transformé les pages HTML existantes en pages PHP.

J'ai également produit une documentation technique sur l'environnement de développement et les normes de codage PHP utilisées.

MISSION 2 : ARCHITECTURE MVC

MVC / MODÈLE-VUE-CONTRÔLEUR / PHP

Restructuration complète de l'application selon le modèle MVC pour séparer la logique, les données et l'affichage.

  • Rédiger une documentation technique sur l'architecture de l'application
  • Rédaction des scripts PHP - HMTL structurant : en-tête, pied de page...
  • Rédaction d'un script MODELE PHP offrant une fonction de connexion à la base de données
  • Rédaction d'un script MODELE PHP offrant des fonctions de gestion des données relatives aux ligues
  • Rédaction d'un script MODELE PHP offrant des fonctions de gestion des données relatives aux départements
  • Rédaction du script VUE PHP - HTML offrant l'affichage des départements
  • Rédaction du script VUE PHP - HTML offrant l'affichage des ligues d'un département
  • Rédaction du script VUE PHP - HTML offrant l'affichage du détail d'une ligue
  • Rédaction des scripts TRAITEMENT PHP offrant les fonctionnalités
