PPE 1
MAISON DES LIGUES

Création d'un site internet et modernisation de l'infrastructure réseau pour la M2L.

MISSION 1 : LE PORTAIL WEB

HTML / CSS / MAQUETTAGE

L'objectif était de créer un portail web statique pour la Maison des Ligues (M2L). Le site devait présenter les services, la structuration des locaux et le personnel.

J'ai également développé un annuaire des ligues (version statique) permettant d'accéder aux informations de 12 ligues réparties sur plusieurs départements (Meurthe-et-Moselle, Meuse, Moselle, Vosges).

MISSION 2 : INFRASTRUCTURE RÉSEAU

CISCO / RÉSEAU / DEVIS

Analyse des besoins pour le passage du réseau en Gigabit. J'ai réalisé l'étude de l'installation physique, le choix du matériel (Câbles Fibre Optique, Commutateurs Cisco SG220-26) et défini le plan d'adressage IP (172.16.0.0/16).

J'ai également établi un devis complet incluant les frais d'installation pour un total de 5 162,69 € TTC.
